The Roach Report

Public service app about restaurant's health inspection

A public service app that ranks restaurant's health inspections with open data from City of Houston.

For this project, I parsed csv data from an Excel file with 36,000 health inspection records and imported the data into my MongoDB database. I built an API to make request from the database, with Node.JS. Than I used Backbone.JS and React.JS to filter and render my filtered and ranked restaurants.

You can preview The Roach Report here. Check out the code on github here.

Tech used in development: Javascript, Backbone.JS, React.JS, Node.JS, MongoDB, HTML & CSS
